Globalización de 3D Slicer

es en fr

La plataforma 3D Slicer es un software de investigación biomédica que cubre la informática de imágenes, el procesamiento de imágenes y la visualización tridimensional. Es soportado por los Institutos Nacionales de Salud de EE.UU. (NIH) y la Alianza Nacional para el Cálculo de Imágenes Médicas (NA-MIC) y 3D Slicer se implementa bajo licencia de código abierto que no restringe su uso. Aunque sus funcionalidades cubren las necesidades de los profesionales y estudiantes de medicina en una amplia variedad de aspectos, una característica importante es que su disponibilidad en diferentes idiomas todavía no se ha implementado (o no se ha implementado completamente).

El objetivo de este proyecto es desarrollar/mejorar el soporte de globalización en 3D Slicer. La globalización del software es el proceso de traducción de software de un idioma de origen a un idioma de destino. El proceso se divide en dos pasos: internacionalización de software (i18n) y localización de software (l10n). La primera es la tarea de diseñar software sin una ubicación específica, es decir, construyéndolo independientemente de cualquier idioma o cultura específica. Este último es el proceso de adaptación de un software para una ubicación específica.
